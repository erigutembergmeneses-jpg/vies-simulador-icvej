# Simulador ICVEJ

**Índice Composto de Visibilidade do Erro Judicial**

Ferramenta interativa para calcular o ICVEJ do seu estado com base em 5 dimensões: Transparência, Mecanismos de Revisão, Velocidade de Correção, Aprendizado Institucional e Prevenção Prospectiva.

## Funcionalidades

- Questionário com 15 perguntas objetivas.
- Cálculo automático do ICVEJ (0–100).
- Comparação com a média nacional (Brasil) e com jurisdições internacionais (Canadá, Reino Unido, EUA, Holanda, Portugal).
- Gráficos de radar e barras.
- Identificação de lacunas e recomendações personalizadas de reforma.

## Como usar

1. Acesse `index.html` em qualquer navegador moderno.
2. Clique em "Iniciar Simulação" e responda às perguntas.
3. Envie o formulário para ver seu ICVEJ.
4. Clique em "Ver relatório completo" para acessar gráficos, comparativos e recomendações.

## Tecnologias

- HTML5, CSS3, JavaScript (ES6)
- [Chart.js](https://www.chartjs.org/) para gráficos (via CDN)
